# Restful Booker Cypress Typescrit Tests

## Objetivo
Testar as ultimas versões do Cypress.

## Instalação
```bash
git clone https://github.com/AllanScala1991/Restful-Booker-Cypress-Test
```

```bash
cd ./Restful-Booker-Cypress-Test
```

```bash
npm install
```

## Como rodar os testes
### Modo headless
```bash
npm run test
```

### Modo open
```bash
npm run open
```
